# 玩法规则

## 目标

从起点「起」画出一条连续路径到终点「终」，并在到达终点前覆盖所有必经格。

## 基础规则

1. 路径只能向上、下、左、右移动，不能斜向移动或跳格。
2. 路径不能进入阻挡格。
3. 同一个格子不能重复进入，因此路径也不能自交。
4. 到达终点时必须已经覆盖所有深色圆点必经格。
5. 进入终点但遗漏必经格时不能继续向外连线，但可以撤回后继续寻找路线。

起点记作第 0 步。“步数”严格等于玩家主动移动次数。每次主动进入相邻格增加一步；进入传送门并从出口出现，虽然路径同时占用入口与出口两个格子，整个传送过程仍只算一步。

## 特殊格

### 单向箭头

进入箭头格后，离开该格时必须沿箭头方向移动。箭头只约束离开方向，不约束进入方向。

### 传送门

相同字母的传送门成对出现。进入其中一个入口后会立即到达另一个出口：

- 入口与出口都计为已占用；
- 传送整体算一步；
- 若出口已经被当前路径占用，则不能进入该传送门。

### 必须转弯

带转弯符号的格子要求进入方向与离开方向垂直，不能直线穿过。

### 限步门

标记 `≤n` 的格子必须在第 `n` 步或之前进入。超过限制后，该格无法进入。

### 挑战印章与顺序标记

- 印章不是通关必需格，但收集指定印章可完成挑战星。
- `①`、`②` 等顺序标记必须按编号顺序经过。

## 三星目标

三颗星彼此独立：

| 星星 | 条件 |
| --- | --- |
| 完成星 | 覆盖所有必经格并合法抵达终点 |
| 最优星 | 使用题目唯一的最短步数完成 |
| 挑战星 | 完成当天额外条件，例如不撤销、限时、收集印章或指定顺序 |

玩家可以无限次尝试。排行榜记录每位玩家当天最优的单次完整成绩，不会将不同尝试的星星、步数和时间拼接。

## 排行规则

每日榜依次比较：星数、步数、活动用时和首次达到该成绩的时间。月度赛季榜累计每天的最佳成绩，依次比较累计星数、总步数和总用时。赛季按北京时间自然月结算并永久保留历史榜单。

## 每日题

- 业务日期按 `Asia/Shanghai` 计算，每天北京时间 00:00 切换。
- 同一日期与生成器版本得到完全相同的谜题。
- 难度从周一到周日逐步提升。
- 服务端求解器验证至少存在两条合法路径，并且最短解唯一。
- 路径型挑战必须只被部分合法路径满足，避免挑战星自动获得。

## 操作方式

- 鼠标或触屏：从当前路径末端拖向相邻格。
- 键盘：聚焦棋盘后使用方向键；也可聚焦格子并按 Enter 或 Space。
- 「撤一步」删除最后一次主动移动。
- 「重新来」清空路径，但正式尝试计时不会归零，且会影响“不撤销”挑战。
